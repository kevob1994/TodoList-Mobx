# TodoList-Mobx

Es una app sencilla, que cuenta con un crud de tareas que permite crear, modificar y eliminar 

## Tecnologías

- React.js
- Mobx
- TypeScript
- Sass
- Ant Design



## Instalación
 
 ```
 git clone https://github.com/kevob1994/TodoList-Mobx.git
 
 npm install / yarn add
 
 npm start
 ```
